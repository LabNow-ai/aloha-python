## Getting started

Refer to[📚 Document & 中文文档](https://aloha-python.readthedocs.io/) for detailed introduction.

```shell
pip install aloha[all]
```

And then:

```python
from aloha.logger import LOG
from aloha.settings import SETTINGS as S
```
